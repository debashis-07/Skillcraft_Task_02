📌 Task 02: Data Cleaning & Preparation

This project was completed as part of my Data Analyst Internship at SkillCraft Technology.

🎯 Objective

The objective of this project is to clean and prepare the Global Superstore Dataset using Python and the Pandas library. The dataset is processed by handling missing values, removing duplicate records, converting date columns into the correct format, and exporting the cleaned dataset for further analysis.

✨ Features

Load the Global Superstore dataset

Display dataset information

Check for missing values

Remove missing values

Remove duplicate records

Convert date columns into DateTime format

Save the cleaned dataset as a new CSV file
